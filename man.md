SYNOPSIS
========
**serve** \[_SCRIPT_] \[**-i**|**\--icon** _ICON_]

DESCRIPTION
===========
Bundle any script as a macOS service.

OPTIONS
=======
**-h**, **\--help**
: Print this brief usage information.

**-v**, **\--version**
: Print version number.

**-i**, **\--icon**
: Apply icon from any image or **.icns** file.


EXAMPLES
========
**serve script.sh \--icon \*.icns**
